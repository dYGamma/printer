# Telegram MFP Bot for Windows

Этот бот превращает компьютер с подключенным USB-принтером/сканером (HP MFP и другие) в сетевое устройство через Telegram.

## Возможности
- 🖨 **Печать**: Поддержка PDF, DOCX, XLSX, PPTX, JPG, PNG.
- 📠 **Сканирование**: Команда `/scan` получает изображение со сканера.
- 🔒 **Безопасность**: Доступ только для разрешенных пользователей (Whitelist).

## Требования
- ОС: **Windows 10/11**
- Python 3.9+
- Установленный Microsoft Office (для печати документов DOCX/XLSX)
- Подключенный и настроенный принтер/сканер (драйверы должны быть установлены)

## Установка

1. **Клонируйте репозиторий**:
   ```powershell
   git clone <URL>
   cd bot-mfp
   ```

2. **Установите зависимости**:
   ```powershell
   pip install -r requirements.txt
   ```

3. **Настройка**:
   Создайте переменные окружения (или отредактируйте `config.py`):
   
   - `TELEGRAM_BOT_TOKEN`: Токен вашего бота от @BotFather.
   - `ALLOWED_USERS`: ID пользователей через запятую (напр. `12345678, 87654321`). Узнать ID можно у бота @userinfobot.

   Пример запуска в PowerShell:
   ```powershell
   $env:TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
   $env:ALLOWED_USERS="12345678"
   python bot/main.py
   ```

## Примечания
- Для печати документов Word/Excel бот использует автоматизацию (COM), поэтому на сервере должен быть установлен Office.
- Для PDF и картинок используется программа, назначенная в системе по умолчанию для действия "Печать". Убедитесь, что файлы открываются и печатаются корректно вручную.
